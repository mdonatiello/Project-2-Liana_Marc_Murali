Project Description

This project analyzed 2075 records on New York State transportation construction projects. The dataset was "NYS Transportation Projects in Your Neighborhood" from the Open NY Dataset. CSV versions of the data were accessed from Kaggle.com. A JSON file for the same dataset was accessed for an API pull into MongoDB.

Significant Finding

New York City was last in terms of total number of projects, total amount of funding and projects in future development. NYC was tied for second to last with Mohawk Valley in terms of projects currently in development, ahead of only the Capital District.
